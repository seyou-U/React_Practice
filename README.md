# React練習 ・ 学習用リポジトリ

## 概要
本リポジトリではtechtrainなどの学習教材を元に学習した内容をハンズオンするためのリポジトリです。

ハンズオンした内容を本リポジトリに更新し、学習および復習のための使用しております。
また、本書以外でも学習用にコーディングしている箇所や内容がございます。

## GitFlow
**mainブランチ**を親ブランチとして、それぞれブランチを作成する運用をしております。
